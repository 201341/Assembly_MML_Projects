Lab 3: ASCII-risks (Asterisks)

----------
DESCRIPTION

Introduction to the MIPS ISA using MARS. Writing a program with several nested loops to print variable-sized ASCII diamonds 
and a sequence of embedded numbers to create a triangle. Introduction to assembly language coding.

----------
FILES

-
Lab3.asm

This file includes the main objective of the lab in an ASM-type file.
Contains assembly language code that will create a triangle using a sequence of numbers and asterisks.

-
README.txt

Describes purpose of Lab3 along with what files will be submitted.
Gives description on how to run lab.

----------
INSTRUCTIONS

This program is intended to be run using the MARS program. Press build
and then play, causing the program to ask for user input.
