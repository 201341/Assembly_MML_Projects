Lab 1: Intro to Logic Simulation

----------
DESCRIPTION

Practice creating truth tables and implementing logic based
on those truth tables, including an introduction to MML.

----------
FILES

-
Lab1.lgi

This file includes the main objective of the lab in an MML-type file. 
Contains Parts A - C along with comments for each part, with each part
meant to show us how input switches, senders, receivers, and boolean
algebra works to create a system of circuits.

-
README.txt

Describes purpose of Lab1 along with what files will be submitted.
Gives description on how to run lab.

----------
INSTRUCTIONS

This program is intended to be run using the MML program. Press play
and use the input switches to observe how the progrm runs.
